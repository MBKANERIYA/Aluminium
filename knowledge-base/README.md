# Aluminium Landing Page

A premium corporate landing page for an Aluminium Manufacturing & Recycling company, built with React and Vite.

## Tech Stack
- Frontend: React (Vite)
- Styling: Vanilla CSS
- Icons: Lucide React

## Directory Structure
- `src/` - Application source code
  - `components/` - React components (`Header.jsx`, `LandingPage.jsx`, `AboutUs.jsx`, `Products.jsx`, `ContactUs.jsx`) and styles (`Header.css`, `LandingPage.css`, `AboutUs.css`, `Products.css`, `ContactUs.css`)
  - `styles/` - CSS files
  - `App.jsx` - Main application entry handling multi-page navigation (`#home`, `#about-page`, `#products-page`, `#contact-page`)
- `knowledge-base/` - Project documentation and rules

## Critical Rules
- Keep existing content exactly the same (mapped from khetarpalmultitrade.com).
- Design aesthetics: Clean, high-contrast, professional corporate Light Industrial design.
- Theme: Modern Premium Industrial Navy, Steel Blue & Aluminum Silver (`#1F3B68` Deep Industrial Navy primary, `#3E5F8A` Steel Blue secondary, `#C8D0D9` Aluminum Silver accents, `#FFFFFF` & `#F8FAFC` section backgrounds, `#162A45` Dark Navy footer, `#111827` primary text, `#475569` secondary text) with premium typography (`Outfit` / `Plus Jakarta Sans` / `Manrope`).
- Rounded cards (`18px` radius), thin `#E5E7EB` borders, soft elevation shadows (`0 10px 30px rgba(17, 24, 39, 0.05)`), high-contrast white text (`#FFFFFF !important`) on solid Navy backgrounds and active states.
